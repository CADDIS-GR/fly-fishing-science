## 📌 이 저장소에 대하여

플라이 낚시의 **"왜(Why)"에 답하는** 과학·이론 자료를 정리합니다.  
출조 기록(fishing-log)이나 기술 메모(fishing-technique)와 달리,  
어류 생태·시각·물리학 등 배경 지식을 체계적으로 축적하는 레퍼런스 저장소입니다.
